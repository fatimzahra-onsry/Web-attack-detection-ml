# attack-detection-data-pipeline

## Project Overview
This project focuses on analyzing and preprocessing cybersecurity attack data collected from multiple CSV and JSON files.

The main objective is to:
- Merge and clean data from different sources
- Handle missing and inconsistent values
- Explore attack patterns
- Visualize important insights

---

## Features
✔ CSV files merging  
✔ JSON files flattening and merging  
✔ Missing values analysis  
✔ Data preprocessing  
✔ Exploratory Data Analysis (EDA)  
✔ Data visualization
